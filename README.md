# Brenna Brocker Portfolio

This is a simple personal portfolio site built with Jekyll and GitHub Pages to share work, interests, and projects in chemical engineering.

## What this site includes
- A short introduction about who Brenna is and what she is interested in
- A projects page highlighting work in process design, optimization, and safety
- A clean, minimal layout that feels professional and easy to read

## Tech Stack
- Jekyll
- Markdown
- GitHub Pages

## Run locally
```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000 in your browser.

## Purpose
This site is a simple way to share engineering work and interests with employers, collaborators, and anyone interested in what Brenna is building.
